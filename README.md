# calculadora-c

Calculadora escrita em C com:

- adição
- subtração
- multiplicação
- divisão

Suporta até *duas* casas decimais de precisão

### Para a compilação e/ou execução, vá para o diretório do repositório instalado localmente

## Compilação (opcional)

`gcc -o main main.c`

## Execução

`./main`
